突破一：精密物理诊断 Precision Signal Diagnostics

振动高频共振包络解调 (HFRT)

解析信号与包络计算公式
z(t) = x(t) + i · H[x(t)]
A(t) = |z(t)| = √(x²(t) + H²[x(t)])
通过 Hilbert 变换提取调制包络波形，滤除背景宽带载波，直接锁死缺陷频率。
  • 特征微弱：早期点蚀冲击能量弱，常规频谱分析中谱线全被背景噪声盖住。
  • 希尔伯特解调：在后端引入希尔伯特变换，构建解析信号，获取瞬时幅值包络。
  • 诊断呈现：前端集成一键切换，对比“原始时域波形”与“包络谱图”，让冲击规律一目了然。
